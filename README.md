# boeidl

Compilador DSL en Rust para los "diseños de registro" (formato BOE) de la AEAT. Define modelos tributarios en ficheros `.boe` legibles y genera código Rust que serializa, deserializa y valida ficheros de posición fija.

> Estado: en desarrollo temprano. V1 objetivo: Modelo 130.

## Desarrollo

```bash
nix develop
cargo test
```
